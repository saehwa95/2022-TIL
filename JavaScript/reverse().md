# reverse()

``` JavaScript
a.reverse()
```

- ```reverse()``` 메서드는 배열의 순서를 반전한다. 
- 첫 번째 요소는 마지막 요소가 되며 마지막 요소는 첫 번째 요소가 된다.


## 예제
``` JavaScript
const a = [1, 2, 3];
console.log(a); // [1, 2, 3]

a.reverse();
console.log(a); // [3, 2, 1]
```
